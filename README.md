# FUNDAMENTAL GO-LANG

## Install
1. download dan install disini [disini](https://golang.org/)
    ```bash
    # cek versi
    go version
    go version go1.15.4 darwin/amd64
    ```


## get started
```bash
mkdir pertama
cd pertama

# create project
go mod init pertama
# result
go: creating new go.mod: module pertama

## running
go run main.go
## hasil
hallo, belajar Golang

## build
go build
./pertama
##hasil
hallo, belajar Golang

go build -o gokil
./gokil
hallo, belajar Golang

# nama package adalah name karena agar executable
```
